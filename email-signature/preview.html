How to install

Apple Mail

  1. Open signature.html in Safari (double-click the file).
  2. Select all (⌘A) and copy (⌘C).
  3. Mail → Settings → Signatures. Create a new signature.
  4. Paste (⌘V) into the signature pane. Uncheck "Always match my default message font".

Outlook (new) / Outlook 365 web

  1. Open preview.html in your browser, scroll to the signature block above.
  2. Select the entire signature (just the table) and copy (⌘C / Ctrl+C).
  3. Outlook → Settings → Mail → Compose and reply → Email signature.
  4. Paste into the editor and save.
Logo not showing in Outlook for Windows?
  1. Push your latest deploy so https://dottacunha.pt/email/logo-white.png goes live.
  2. Open signature.html, find the long src="data:image/png;base64,..." string, and replace it with src="https://dottacunha.pt/email/logo-white.png".
  3. Re-paste into Outlook.
